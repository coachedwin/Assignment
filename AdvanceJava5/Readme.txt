Steps to run this Application :

1.At first APIApplication should be run as java application with main class Application2Application
2.Then we have to run loginApplication on server(to login you first have to register yourself by clicking on create account. After successful registration you will be redirected to login page)
3.You have to add Authors names in database manually so that it can be fetched by API(query : insert into demo.authors values(1,"James Gosling");)
4.Now you can add, delete, edit book lists

Note: database/schema = demo
